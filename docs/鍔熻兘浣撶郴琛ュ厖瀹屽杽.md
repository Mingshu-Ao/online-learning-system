# 功能体系补充完善

## 1. 学员端功能

学员端不仅要能“看课”，还要形成完整学习闭环。

核心功能包括：

1. **课程学习**
   - 课程列表、课程详情、章节目录。
   - 视频播放、倍速播放、断点续播。
   - 文档课件预览与下载权限控制。
   - 学习进度自动记录。
   - 视频播放防刷进度，例如限制直接拖动超过已学习位置。
2. **互动学习**
   - 视频中弹题。
   - 章节测验。
   - 课程期末测验。
   - 提交答案后自动判题。
   - 主观题支持教师批改或 AI 辅助评分。
3. **错题本**
   - 自动收录错题。
   - 按课程、章节、知识点分类。
   - 支持重做、标记已掌握。
   - 与知识图谱联动，推荐前置知识点。
4. **学习轨迹**
   - 今日学习时长。
   - 连续学习天数。
   - 课程完成率。
   - 测验正确率。
   - 专注时长统计。
   - 学习日历热力图。
5. **证书系统**
   - 完成课程后生成电子证书。
   - 证书编号唯一。
   - 支持证书可视化展示。
   - 管理端可审核证书发放规则。
6. **线上自习室**
   - 进入自习室。
   - 分配虚拟座位。
   - 设置番茄钟。
   - 实时查看同房间成员状态。
   - 完成专注后生成打卡记录。
   - 支持学习排行榜。
7. **AI 智能助教**
   - 文本提问。
   - 上传错题截图。
   - 上传手写草稿。
   - OCR 识别题目。
   - 返回解题思路、相关知识点、推荐视频片段。
   - 对高频问题生成班级 FAQ。

## 2. 教师端功能

教师端重点是“教学资源管理 + 学情分析 + 作业测评”。

核心功能包括：

1. **课程管理**
   - 创建课程。
   - 编辑课程介绍。
   - 设置课程封面。
   - 设置课程上下架状态。
   - 管理课程章节。
2. **资源管理**
   - 视频分片上传。
   - 文档上传。
   - 资源转码状态查看。
   - 资源权限设置。
   - 资源与章节绑定。
3. **题库管理**
   - 单选题、多选题、判断题、填空题、简答题。
   - 富文本题干。
   - 图片题干。
   - 题目知识点标签。
   - 难度等级。
   - 批量导入题目。
4. **试卷管理**
   - 手动组卷。
   - 按知识点、难度、题型自动组卷。
   - 设置考试时间。
   - 设置是否允许重做。
   - 设置及格线。
5. **作业与测验批改**
   - 客观题自动判分。
   - 主观题人工批改。
   - AI 辅助生成评分建议。
   - 查看学生答题详情。
6. **学情监控**
   - 班级学习进度。
   - 视频完播率。
   - 测验平均分。
   - 知识点薄弱分布。
   - 未学习学生提醒。
   - 异常学习行为检测。

------

## 3. 管理端功能

管理端重点是“平台治理”。

核心功能包括：

1. **用户管理**
   - 学生、教师、管理员账号管理。
   - 账号启用/禁用。
   - 密码重置。
   - 用户导入导出。
2. **RBAC 权限管理**
   - 角色管理。
   - 权限管理。
   - 菜单权限。
   - 按钮权限。
   - 接口权限。
3. **课程审核**
   - 教师提交课程。
   - 管理员审核上下架。
   - 审核意见记录。
4. **系统公告**
   - 发布公告。
   - 指定可见范围。
   - 首页公告展示。
5. **日志审计**
   - 登录日志。
   - 操作日志。
   - 资源访问日志。
   - 异常日志。
   - 管理员敏感操作追踪。
6. **平台运营数据**
   - 活跃用户数。
   - 学习总时长。
   - 课程完成率。
   - 自习室使用情况。
   - AI 助教调用量。

## 4. 创新功能补充

建议把创新点明确写成三个方向。

### 创新点一：线上自习室实时监督

系统通过 WebSocket + Redis 实现虚拟自习室：

- 用户进入房间后分配虚拟座位。
- Redis 保存当前在线座位状态。
- WebSocket 实时广播状态变化。
- 支持专注中、小憩中、已离开、异常断线等状态。
- 番茄钟完成后生成专注记录。
- 形成自习排行榜和打卡海报。

### 创新点二：多模态 AI 智能助教

系统通过独立 AI 微服务支持：

- 题目截图 OCR。
- 手写内容识别。
- 图文混合题目解析。
- 知识点提取。
- 解题步骤生成。
- 推荐相关课程片段。
- 生成错因分析。

### 创新点三：基于知识图谱的个性化学习路径

系统将课程知识点建模为图结构：

- 知识点之间存在前置依赖关系。
- 测验错题映射到知识点。
- 系统回溯薄弱知识点的前置节点。
- 推荐基础视频、题目和学习路径。
- 实现“不是哪里错就只学哪里”，而是追溯真正薄弱环节。
